Early words

Which words do children say, and at what age? We analyze data from the Stanford wordbank database using the wordbankr package to find out.

We have data on children from 16 to 30 months of age.

plot <- ggplot(words, aes(x = age, y = produces, color = lemma)) + 
  geom_smooth(se = FALSE, lwd = .5) +
  geom_point() 
  
plot + scale_color_paletteer_d("nationalparkcolors::Saguaro")
